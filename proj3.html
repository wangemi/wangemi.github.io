UMessage

LANGUAGE: Java
CLASS PROJECT: Data Structures & Parallelism
PARTNER PROJECT
DURATION: 4.5 weeks
AUTUMN 2019

UMessage is a texting application that suggests words for users as they are typing.


The Focus

This project focused on learning how to implement many dictionary implementations like AVLTrees, chaining hash tables, move to front lists, and hash tries. I also implemented common sorting algorithms like heap sort, quick sort, and top k sort.
