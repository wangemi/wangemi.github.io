Jamboree Bot

LANGUAGE: Java
CLASS PROJECT: Data Structures & Parallelism
PARTNER PROJECT
DURATION: 4.5 weeks
AUTUMN 2019

Jamboree Bot is a chess bot that uses parallelism to calculate best moves against an opponent.


The Focus

This project focused on how to correctly implement parallel code using Java's Fork/Join Framework. I explored how to convert different sequential programs into parallel ones, and the impact of factors like cutoffs and number of processors on performance.
